# Product Context

## Project Overview

A cross-browser extension that detects checkout pages, detects whether a user is about to spend money on an unethical company or product, encourages them to spend money on alternatives, and collects revenue through referral codes and affiliate links.

## Core Features

1. Checkout Detection
   - Pattern-based URL matching
   - DOM analysis for checkout indicators
   - Caching for performance optimization
   - Cross-browser compatible detection methods

2. Ethical Analysis
   - Company/product ethics evaluation
   - Alternative product suggestions
   - Ethical shopping recommendations

3. User Interface
   - Non-intrusive notifications
   - Clear alternative suggestions
   - Consistent experience across browsers
   - Responsive popup design

4. Revenue Generation
   - Affiliate link integration
   - Referral code management
   - Cross-browser tracking support

## Browser Support

1. Chrome/Chromium
   - Primary development platform
   - Full feature support
   - Chrome Web Store distribution

2. Firefox
   - Full feature parity with Chrome
   - Firefox-specific manifest configuration
   - Mozilla Add-ons distribution

3. Edge
   - Chromium-based compatibility
   - Edge-specific UI adjustments
   - Microsoft Store distribution

4. Safari
   - Core feature support
   - Safari Web Extensions compatibility
   - App Store distribution requirements

## Technical Requirements

1. Cross-Browser Compatibility
   - Browser-agnostic code structure
   - Feature detection and fallbacks
   - Unified messaging system
   - Consistent storage handling

2. Performance
   - Fast checkout detection
   - Efficient DOM operations
   - Minimal resource usage
   - Quick UI responses

3. Security
   - Safe DOM manipulation
   - Secure message passing
   - Private data handling
   - Cross-origin safety

4. Maintainability
   - Clear code organization
   - Comprehensive documentation
   - Automated testing
   - Version control best practices
