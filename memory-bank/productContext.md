# Product Context

## Project Overview

A cross-browser extension that detects checkout pages, detects whether a user is about to spend money on an unethical company or product, encourages them to spend money on alternatives, and collects revenue through referral codes and affiliate links.

## Core Memory Bank Files

- `productContext.md`: Describes the project's vision, goals, and overall context.
- `activeContext.md`: Tracks the current session's state, goals, and recent changes.
- `progress.md`: Documents completed work, next steps, and overall project progress.
- `decisionLog.md`: Records key architectural decisions, their rationale, and implementation details.

Additional files may be created as needed to support the project's development.
